About Program

This application is a web application developed using Spring Boot and RESTful services. You can perform various operations through the endpoints listed below.

User Operations

  • GET - List Users: /users/
  • GET - User Details: /users/{id}
  • PUT - Update User: /users/{id}
  • DELETE - Delete User: /users/{id}
  • POST - Add New User: /users/

Post Operations

  • GET - List Posts: /posts/
  • GET - User's Posts: /posts/users/{id}
  • GET - User's Post Order: /posts/orderuser
  • GET - Sort User's Posts by Category: /posts/ordercategory
  • GET - Sort Posts by Category: /posts/category/{id}
  • GET - Sort User's Posts by Publish Date: /posts/orderpublish
  • PUT - Update Post: /posts/{id}
  • DELETE - Delete Post: /posts/{id}
  • POST - Add New Post: /posts/create

Category Operations

  • GET - List Categories: /categories/
  • GET - Category Details: /categories/{id}
  • PUT - Update Category: /categories/{id}
  • DELETE - Delete Category: /categories/{id}
  • POST - Add New Category: /categories/
  • GET - List Categories by Name: /categories/name
